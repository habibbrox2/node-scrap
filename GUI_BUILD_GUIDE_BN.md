# Brox Scraper GUI - বিল্ড এবং ইনস্টলেশন গাইড

## GUI অ্যাপ তৈরি করার ধাপগুলি

### পদক্ষেপ ১: নতুন ডিপেন্ডেন্সি ইনস্টল করুন

```bash
npm install
```

এটি electron এবং electron-builder ইনস্টল করবে।

### পদক্ষেপ ২: আইকন সেটআপ করুন (ঐচ্ছিক)

একটি `assets` ফোল্ডার তৈরি করুন এবং একটি 256x256 পিক্সেল PNG আইকন রাখুন:

```bash
mkdir assets
# এখানে আপনার icon.png রাখুন (256x256 পিক্সেল)
```

### পদক্ষেপ ৩: ডেভেলপমেন্টে পরীক্ষা করুন

GUI অ্যাপ চালু করুন:

```bash
npm run electron-start
```

বা development মোডে (livereload সহ):

```bash
npm run electron-dev
```

### পদক্ষেপ ৪: ইনস্টলেবল এক্সিকিউটেবল তৈরি করুন

#### সম্পূর্ণ ইনস্টলার (NSIS - সুপারিশকৃত):

```bash
npm run build:gui
```

এটি `dist/` ফোল্ডারে একটি ইনস্টলার তৈরি করবে:
- `Brox Scraper Setup 1.0.0.exe` - সম্পূর্ণ ইনস্টলার
- `Brox Scraper 1.0.0 Setup.exe` - অপরিবর্তনীয় ইনস্টলার

#### পোর্টেবল সংস্করণ (কোন ইনস্টলেশন নেই):

```bash
npm run build:gui-win-portable
```

এটি তৈরি করবে:
- `Brox Scraper-1.0.0-portable.exe` - সরাসরি চালানোর জন্য

## ফাইল স্ট্রাকচার

নতুন ফাইল যুক্ত করা হয়েছে:

- **electron-main.js** - Electron মূল প্রক্রিয়া
- **preload.js** - নিরাপত্তা প্রিলোড স্ক্রিপ্ট
- **installer-config.js** - বিল্ডার কনফিগ (package.json-এ অন্তর্ভুক্ত)

## ফিচার

- 🎯 **সম্পূর্ণ GUI** - Express ড্যাশবোর্ড Electron উইন্ডোতে দেখা যায়
- 💾 **স্বয়ংক্রিয় সার্ভার ম্যানেজমেন্ট** - Electron অ্যাপ চালু হলে সার্ভার শুরু হয়
- 📦 **সহজ ইনস্টলেশন** - NSIS ইনস্টলার
- 🌐 **মাল্টি-ভাষা মেনু** - বাংলা মেনু অপশন
- ⚡ **পোর্টেবল সংস্করণ** - কোন ইনস্টলেশন প্রয়োজন নেই

## ট্রাবলশুটিং

### পোর্ট 9999 ইতিমধ্যে ব্যবহৃত হলে

electron-main.js-এ PORT পরিবর্তন করুন।

### আইকন দেখা যাচ্ছে না

`assets/icon.png` নিশ্চিত করুন যে বিদ্যমান এবং 256x256 পিক্সেল।

### বিল্ড ব্যর্থ হচ্ছে

```bash
rm -r node_modules dist
npm install
npm run build:gui
```

## পরবর্তী পদক্ষেপ

1. আপনার লোগো দিয়ে `assets/icon.png` প্রতিস্থাপন করুন
2. `package.json`-এ সংস্করণ আপডেট করুন
3. NSIS ইনস্টলার কাস্টমাইজ করার জন্য `package.json` এর `build.nsis` অংশ সম্পাদনা করুন
