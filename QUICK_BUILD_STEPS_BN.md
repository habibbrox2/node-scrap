# Build এবং Test করার Quick Steps

## 🚀 দ্রুত শুরু করুন

### স্টেপ ১: প্রজেক্ট ফোল্ডার খুলুন
```powershell
cd h:\Web\scrap_data
```

### স্টেপ २: সব নির্ভরতা ইনস্টল করুন
```bash
npm install
```
*(এটি ৫-১০ মিনিট সময় লাগবে)*

---

## 🔨 Build করুন

### অপশন ১: সম্পূর্ণ ইনস্টলার (সুপারিশকৃত)
```bash
npm run build:gui-win
```
**আউটপুট:** `dist/Brox Scraper 1.0.0.exe` ইনস্টলার

---

### অপশন २: পোর্টেবল এক্সিকিউটেবল
```bash
npm run build:gui-win-portable
```
**আউটপুট:** `dist/Brox Scraper 1.0.0-portable.exe` (কোন ইনস্টলেশন প্রয়োজন নেই)

---

### অপশন ३: স্ট্যান্ডঅ্যালোন .EXE
```bash
npm run build:windows
```
**আউটপুট:** `dist/brox-scraper.exe` (একক ফাইল)

---

## 🧪 Test করুন

### স্টেপ १: তৈরি ফাইল খুলুন
```bash
cd dist
.\Brox\ Scraper\ 1.0.0.exe
```

### স্টেপ २: ইনস্টলেশন সম্পন্ন করুন
- Setup উইজার্ড অনুসরণ করুন
- ডেস্কটপ শর্টকাট চেক করুন
- "ইনস্টল" ক্লিক করুন

### স्टेপ ३: অ্যাপ চালান
- Start Menu থেকে "Brox Scraper" খুঁজুন
- অথবা ডেস্কটপ শর্টকাট চালান

### স্টেপ ४: নতুন ফিচার টেস্ট করুন
1. অ্যাপ খোলার পর মেনু খুলুন
2. "ফাইল" → "আপডেট চেক করুন" ক্লিক করুন
3. আপডেট চেক সম্পন্ন হওয়ার জন্য অপেক্ষা করুন
