# Brox Scraper GUI - সম্পূর্ণ সেটআপ এবং বিল্ড গাইড

## 🎯 উদ্দেশ্য
আপনার Brox Scraper Node.js অ্যাপ্লিকেশনকে একটি **ইনস্টলেবল Windows GUI অ্যাপ্লিকেশন** এ রূপান্তরিত করা।

## ✅ যা তৈরি হয়েছে

### নতুন ফাইলগুলি:

| ফাইল | উদ্দেশ্য |
|------|--------|
| `electron-main.js` | Electron অ্যাপ্লিকেশনের মূল ফাইল |
| `preload.js` | নিরাপত্তা প্রিলোড স্ক্রিপ্ট |
| `package.json` | আপডেট করা (electron ডিপেন্ডেন্সি যুক্ত) |
| `build-gui.bat` | Windows এ বিল্ড করার জন্য স্বয়ংক্রিয় স্ক্রিপ্ট |
| `build-gui.sh` | Linux/Mac এ বিল্ড করার জন্য স্ক্রিপ্ট |
| `generate-icon.bat` | আইকন জেনারেট করার স্ক্রিপ্ট |
| `GUI_BUILD_GUIDE_BN.md` | বিস্তারিত বাংলা গাইড |
| `QUICK_START_BN.md` | দ্রুত শুরু করার গাইড |

## 🚀 দ্রুত শুরু (৩ ধাপে)

### ধাপ ১: আইকন তৈরি করুন (ঐচ্ছিক)

```bash
generate-icon.bat
```

এটি `assets/icon.png` তৈরি করবে। অথবা আপনার নিজের আইকন রাখুন।

### ধাপ ২: বিল্ড শুরু করুন

ডাবল-ক্লিক করুন:
```
build-gui.bat
```

বা কমান্ড প্রম্পটে:
```cmd
npm run build:gui
```

### ধাপ ৩: ইনস্টলার খুঁজুন এবং চালান

```
dist/Brox Scraper Setup 1.0.0.exe
```

## 📋 বিস্তারিত নির্দেশাবলী

### প্রি-রিকুইজিট

✅ **Node.js ইনস্টল করা আছে?**
```bash
node --version
npm --version
```

❌ যদি না থাকে:
- Windows: https://nodejs.org/ থেকে ডাউনলোড করুন
- চেক করুন যে `node` এবং `npm` কমান্ড কাজ করে

---

### স্টেপ-বাই-স্টেপ প্রক্রিয়া

#### পর্যায় 1: প্রাথমিক সেটআপ

```bash
# 1. প্রজেক্ট ডিরেক্টরিতে যান
cd h:\Web\scrap_data

# 2. npm dependencies ইনস্টল করুন
npm install
```

এটি ইনস্টল করবে:
- `electron` - ডেস্কটপ অ্যাপ ফ্রেমওয়ার্ক
- `electron-builder` - ইনস্টলার তৈরির টুল
- অন্যান্য প্রয়োজনীয় প্যাকেজ

#### পর্যায় 2: আইকন সেটআপ (ঐচ্ছিক কিন্তু সুপারিশকৃত)

##### অপশন A: স্বয়ংক্রিয় আইকন জেনারেশন
```bash
generate-icon.bat
```

##### অপশন B: কাস্টম আইকন ব্যবহার করুন
1. একটি 256x256 পিক্সেল PNG ছবি প্রস্তুত করুন
2. `assets/icon.png` এ সংরক্ষণ করুন

#### পর্যায় 3: ডেভেলপমেন্টে পরীক্ষা করুন (ঐচ্ছিক)

```bash
npm run electron-start
```

এটি আপনার GUI অ্যাপ খুলবে। আপনি:
- পরীক্ষা করতে পারেন যে সবকিছু কাজ করছে
- ড্যাশবোর্ড ব্যবহার করতে পারেন

বন্ধ করতে: `Ctrl+C`

#### পর্যায় 4: ইনস্টলেবল এক্সিকিউটেবল বিল্ড করুন

**বিকল্প A: সম্পূর্ণ ইনস্টলার (সুপারিশকৃত)**

```bash
npm run build:gui
```

আউটপুট:
- `dist/Brox Scraper Setup 1.0.0.exe` - সম্পূর্ণ ইনস্টলার
- ~150 MB

**বিকল্প B: পোর্টেবল এক্সিকিউটেবল**

```bash
npm run build:gui-win-portable
```

আউটপুট:
- `dist/Brox Scraper-1.0.0-portable.exe` - কোন ইনস্টলেশন প্রয়োজন নেই
- ~150 MB

#### পর্যায় 5: বিতরণ

```bash
# dist ফোল্ডারে যান
cd dist

# ফাইল দেখুন
dir *.exe
```

এখন আপনি পাবেন:
- ইনস্টলেশন প্রস্তুত `.exe` ফাইল
- এটি যেকোনো সময় শেয়ার এবং চালানো যায়

## 🔄 কীভাবে এটি কাজ করে

### আর্কিটেকচার

```
┌─────────────────────────────────────┐
│  Electron Main Process              │
│  (electron-main.js)                 │
│  ┌──────────────────────────────┐   │
│  │ • উইন্ডো ম্যানেজমেন্ট        │   │
│  │ • সার্ভার স্পন টার্ম        │   │
│  │ • মেনু ও IPC                 │   │
│  └──────────────────────────────┘   │
└─────────────────────────────────────┘
         │
         ├─────────────────────────┐
         │                         │
    ┌────▼────┐           ┌──────▼──────┐
    │ Express │           │   Browser   │
    │ Server  │           │   Window    │
    │ Port    │◄─────────►│ localhost:  │
    │ 9999    │           │ 9999        │
    └─────────┘           └─────────────┘
         │
         ├─── scrapers, database, API
         └─── cache, cron jobs
```

### প্রক্রিয়াপ্রবাহ

1. ব্যবহারকারী `.exe` চালান
2. Electron মূল প্রক্রিয়া শুরু হয়
3. Node.js সার্ভার স্বয়ংক্রিয়ভাবে শুরু হয়
4. Express ড্যাশবোর্ড পোর্ট 9999 এ চলে
5. Electron উইন্ডো খোলে এবং `localhost:9999` লোড করে
6. সবকিছু একটি অ্যাপ হিসাবে কাজ করে

`package.json` খুলুন এবং সম্পাদনা করুন:

```json
{
  "name": "my-app-name",
  "productName": "My App Display Name",
  ...
}
```

### পোর্ট পরিবর্তন করুন

`electron-main.js` খুলুন এবং খুঁজুন:

```javascript
const SERVER_PORT = 9999;  // ← এটি পরিবর্তন করুন
```

### মেনু ভাষা পরিবর্তন করুন

`electron-main.js` তে `createApplicationMenu()` ফাংশন সম্পাদনা করুন।

## 🐛 ট্রাবলশুটিং

### সমস্যা: "npm command not found"

**সমাধান:**
- Node.js পুনরায় ইনস্টল করুন
- কমান্ড প্রম্পট পুনরায় খুলুন
- `node --version` চেক করুন

### সমস্যা: বিল্ড দীর্ঘ সময় নেয়

**কারণ:** 
- প্রথম বার electron-builder ডাউনলোড করে (~200MB)

**সমাধান:**
- ধৈর্য রাখুন (5-10 মিনিট)
- দ্রুত ইন্টারনেট ব্যবহার করুন
- অন্য অ্যাপ বন্ধ করুন

### সমস্যা: "Port 9999 already in use"

**সমাধান:**
- `electron-main.js` তে `SERVER_PORT` পরিবর্তন করুন
- অথবা পুরানো প্রক্রিয়া বন্ধ করুন:
  ```bash
  netstat -ano | findstr :9999
  taskkill /PID <PID> /F
  ```

### সমস্যা: আইকন অদৃশ্য

**সমাধান:**
- `assets/icon.png` যা 256x256 পিক্সেল PNG থাকা নিশ্চিত করুন
- পুনরায় বিল্ড করুন

### সমস্যা: ইনস্টলেশন বিফল হচ্ছে

**সমাধান:**
```bash
# পরিষ্কার করুন এবং পুনরায় চেষ্টা করুন
rmdir /s dist
npm install
npm run build:gui
```
