# Quick Start - GUI Windows App

## ধাপে ধাপে নির্দেশনা

### ৩০ সেকেন্ডে শুরু করুন:

1. **build-gui.bat ক্লিক করুন**
   ```
   Windows Explorer এ ডাবল-ক্লিক করুন
   ```
   বা কমান্ড প্রম্পটে:
   ```cmd
   build-gui.bat
   ```

2. **অপেক্ষা করুন** (প্রথম বার ৫-১০ মিনিট লাগতে পারে)

3. **dist ফোল্ডারে ইনস্টলার খুঁজুন**
   ```
   dist/Brox Scraper Setup 1.0.0.exe
   ```

4. **ইনস্টলার চালান এবং ইনস্টল করুন**

## বিভিন্ন অপশন

### পূর্ণ ইনস্টলার (সুপারিশকৃত)
```bash
npm run build:gui
```
ফলাফল: `dist/Brox Scraper Setup 1.0.0.exe`
- সম্পূর্ণ ইনস্টলেশন অভিজ্ঞতা
- শুরু মেনু শর্টকাট
- প্রোগ্রাম সরান অপশন

### পোর্টেবল সংস্করণ (কোন ইনস্টলেশন নেই)
```bash
npm run build:gui-win-portable
```
ফলাফল: `dist/Brox Scraper-1.0.0-portable.exe`
- সরাসরি চালান
- কোন ইনস্টলেশন প্রয়োজন নেই
- USB ড্রাইভে রাখুন এবং যেকোনো জায়গা থেকে চালান

### ডেভেলপমেন্ট মোডে পরীক্ষা
```bash
npm install
npm run electron-start
```

## প্রয়োজনীয়তা

- **Windows 7+** (64-bit)
- **Internet**: প্রথম রান এবং স্ক্র্যাপিং এর জন্য
- স্থান: ~200MB

## সমস্যা সমাধান

### "Node.js পাওয়া যায় না"
Node.js ইনস্টল করুন: https://nodejs.org/

### "পোর্ট 9999 ইতিমধ্যে ব্যবহৃত"
`electron-main.js` খুলুন এবং `SERVER_PORT` পরিবর্তন করুন

### "বিল্ড দীর্ঘ সময় নিচ্ছে"
এটি স্বাভাবিক (প্রথম বার 5-10 মিনিট)। ইন্টারনেট সংযোগ দ্রুত হলে দ্রুত হবে।

## অগ্রগতি ট্র্যাক করুন

স্ক্রিপ্ট প্রতিটি পর্যায়ে প্রগতি দেখাবে:
- [1/4] ডিপেন্ডেন্সি ইনস্টল
- [2/4] Assets সেটআপ
- [3/4] বিল্ড প্রক্রিয়া
- [4/4] সম্পন্ন

## ইনস্টলেশন অভিজ্ঞতা

ব্যবহারকারীরা যা দেখবেন:

1. **স্বাগত স্ক্রিন** - পরবর্তী ক্লিক করুন
2. **ইনস্টল পথ নির্বাচন** - ডিফল্ট সুপারিশ করা হয়
3. **শুরু মেনু শর্টকাট** - স্বয়ংক্রিয়ভাবে তৈরি হয়
4. **ইনস্টলেশন সম্পূর্ণ** - অ্যাপ চালু করুন

## আনইনস্টল

Windows Settings > Apps > Brox Scraper > Uninstall
