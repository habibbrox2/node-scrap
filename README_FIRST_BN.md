# READ ME FIRST - GUI Setup সম্পন্ন! 🎉

আপনার Brox Scraper অ্যাপ্লিকেশন এখন **ইনস্টলেবল Windows GUI অ্যাপ** হিসাবে প্রস্তুত!

---

## 🚀 ৩০ সেকেন্ডে শুরু করুন

```bash
build-gui.bat
```

এটিই! স্ক্রিপ্ট বাকি সবকিছু করবে। ৫-১০ মিনিট অপেক্ষা করুন এবং `dist/` ফোল্ডারে `.exe` পাবেন।

## 🎯 প্রয়োজনীয়তা

- **Windows 7+** (64-bit)
- **Node.js** ইনস্টল করা - [nodejs.org](https://nodejs.org/)
- **Internet** - বিল্ডিং এর সময়

---

## 🔥 দ্রুত কমান্ড

```bash
# সিস্টেম চেক করুন (ঐচ্ছিক)
system-check.bat

# বিল্ড করুন (প্রধান কমান্ড)
build-gui.bat

# অথবা ম্যানুয়ালি
npm install
npm run build:gui

# ডেভেলপমেন্টে পরীক্ষা করুন
npm run electron-start
```

## 📞 পরবর্তী পদক্ষেপ

**এক্ষুনি শুরু করতে:**

1. টার্মিনাল খুলুন
2. এই ফোল্ডারে যান
3. চালান: `build-gui.bat`
4. অপেক্ষা করুন (5-10 মিনিট)
5. `dist/` ফোল্ডারে `.exe` খুঁজুন
6. ইনস্টল করুন এবং উপভোগ করুন!
